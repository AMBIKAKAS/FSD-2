Experiment No. 5
Optimize Frontend Performance using Lazy Loading
1. Aim
To improve the performance of a React application by loading components only when required using lazy loading and code splitting.


------------------------------------------------------------------------------

2. Theory

Frontend performance refers to how fast a web application loads and responds to user actions. Large JavaScript bundles slow down applications.

Lazy loading loads components only when they are needed, reducing the initial bundle size and improving load speed.

---------------------------------------------------------------------------------------------

3. Learning Outcomes

After completing this experiment, the learner will be able to:

Understand the concept of frontend performance and the impact of large JavaScript bundles on application speed.

Explain the working and importance of lazy loading in modern web applications.

Implement lazy loading to optimize React applications and reduce initial load time.

Analyze how code splitting improves responsiveness and overall user experience.
